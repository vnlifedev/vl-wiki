# Выдержка

**Выдержка**

**Выдержка** — это функция, которая позволяет выдерживать ваши напитки в **бочках**. При выдержке зелье получает дополнительные эффекты и значения в зависимости от рецепта. Процесс выдержки является опциональным для каждого зелья.

**Создание бочки**

Чтобы выдержать напиток, необходимо сначала создать бочку. Бочки создаются путем установки **таблички** на набор блоков с надписью **Barrel**.

**Бочки**

После того как напитки помещены в бочки, остается только ждать! 1 “год” равен 1 дню в игре. Если переборщить с выдержкой или использовать неправильный вид древесины, напиток, скорее всего, испортится.
