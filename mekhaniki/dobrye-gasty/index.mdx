# Добрые Гасты

## Гасты - бомбардировщики

Если вы сядете на доброго гаста, возьмете в руки динамит и нажмете на кнопку Q, то вы сбросите динамит вниз. Динамит взорвется через несколько секунд независимо на земле ли он или нет. Мощность динамита урезаны при сбросе.

{% embed url="<https://www.youtube.com/watch?v=IKKKkCb-GsU>" %}

{% hint style="warning" %}
Сброс бомб без причины на жилые участки, территории, фермы и т.д. наказуемо. По умолчанию сброс бомб на сервере отключены, они будут включены при конфликтах. Все сбросы динамита с гастов логируются.
{% endhint %}
