# Emote Craft

## [СКАЧАТЬ](https://modrinth.com/plugin/emotecraft)

## Как использовать

* Нажмите **B**, чтобы открыть колесо выбора эмоций.
* Установите свои эмоции, поместив их в папку:

```
.minecraft/emotes
```

* Либо просто нажмите **«Открыть папку эмоций»** в экране настроек мода.
