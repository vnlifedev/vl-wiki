# Система Идей

Система идей - простая система, которая поможет игрокам более быстрее предлагать что-то построить, попросить что-то добыть, провести ивент и т.д.

Чтобы открыть GUI меню нужно ввести команду `/ideas`

### Команды

/idea create категория описание \[награда] - создать задание

/idea take - \id взять идею в работу

/idea complete \id - завершить идею

/idea cancel \id - отказаться от идеи

/idea list категория|статус|мои|задачи

/idea view \id - просмотр идеи

## Дискорд Бот Винж

Винж - дискорд бот, который работает только когда запущен сервер засчет серверного плагина.

Для идей он может выполнять эти команды:

/ideas list \page:(страница) - список идей
